## 4. Lua VM
C++ 프로그램에서 Lua 인터프리터를 실행하려면 먼저 Lua VM(virtual machine)을 생성해야 합니다.  
```cpp
  // Lua VM 생성
  lua_State* pLuaState = luaL_newstate();
```
Lua State는 Lua VM 입니다.  
Lua State는 하나의 독립된 실행 환경입니다.  
Lua State는 자신만의 스택, 가비지 컬렉터, 메모리공간을 가집니다.
