| 시스템 V AMD64 ABI           | GNU/리눅스, BSD, OS X (GCC, 인텔 C++ 컴파일러)                                   | RDI, RSI, RDX, RCX, R8, R9, XMM0–7   | RTL (C)              | 호출자                |



```cpp
int x = 10; // 1번 줄 강조
int y = 20;
x = x + y; // 3번 줄 강조
return x;  // 4번 줄 강조
```


```assembly
0x00ED1270  push        14h                     stack에 20 push
0x00ED1272  push        0Ah                     stack에 10 push
0x00ED1274  call        Test2 (0ED1110h)        stack에 전달인자를 push 한 다음 함수를 call 한다.
```
