## Heap
Heap 메모리영역은 메모리 동적할당에 사용된다.  
Stack과는 달리 사용자가 메모리를 해제하기 전까지는 메모리가 계속 유지된다.  

프로세스에서 메모리를 사용하려면 먼저 가상메모리를 reserve한 다음, 가상메모리의 page를 commit해야 한다.  
Heap도 마찬가지인데, 프로세스의 Heap Manager가 가상메모리를 최소 64KB 단위로 reserve하여 관리하고, 사용자에게는 최소 16B 단위로 쪼개어 할당해준다.  

## Default Heap
프로세스가 생성될 때 생성되는 Heap을 Default Heap이라고 한다.  
사용자가 Heap을 추가로 생성할 수는 있지만 권장되지 않는다.  
Heap(Default Heap)의 최초 크기는 1MB 이다.  
최초 크기는 1MB지만 모두 reserve 되어있고, Heap을 실제로 사용할 때 필요한 만큼 commit 해서 사용한다.  
Heap은 Stack과 달리 최대크기가 정해져있지 않고 계속해서 자동으로 늘어난다.  

## Low Fragmentation Heap(LFH, 저단편화 힙)
LFH는 Heap의 메모리가 파편화되는 것을 최소화하기위해 설계되었다.  
그리고 멀티스레드 환경에서 Heap의 Lock을 최소화 하기위한 최적화 기능도 들어가 있다.  
프로세스의 Default Heap은 LFH 이다.  

LFH는 메모리 파편화를 피하기 위해 메모리를 bucket이라고 하는 미리 정해진 크기의 블록으로 나누어 관리한다.  
프로세스가 Heap에서 메모리를 할당하려고 하면, LFH는 필요한 크기를 담을 수 있는 최소 크기의 bucket에서 블록을 하나 가져와 할당해준다.  
블록의 최소 크기는 8bytes 이다.  

bucket은 크기별로 나눠져 있는데,  
첫번째 bucket은 8bytes 블록들을 가지며 1 ~ 8bytes 크기의 메모리를 할당할때 사용하고,  
두번째 bucket은 16bytes 블록들을 가지며 9 ~ 16bytes 크기의 메모리를 할당할때 사용하고,  
이런식으로 계속되다가  
32번째 bucket은 256bytes 블록들을 가지며 249 ~ 256bytes 크기의 메모리를 할당할때 사용하고,  
마지막 128번째 bucket은 16384bytes 블록들을 가지며 15873 ~ 16384bytes 크기의 메모리를 할당할 때 사용한다.  

아래는 bucket 번호와 granularity, 할당가능한 메모리 크기를 표로 나타낸 것이다.

| Buckets | Granularity | Range        |
| ------- | ----------- | ------------ |
| 1–32    | 8           | 1–256        |
| 33–48   | 16          | 257–512      |
| 49–64   | 32          | 513–1,024    |
| 65–80   | 64          | 1,025–2,048  |
| 81–96   | 128         | 2,049–4,096  |
| 97–112  | 256         | 4,097–8,194  |
| 113–128 | 512         | 8,195–16,384 |


멀티스레드 환경에서의 Heap의 Lock을 최소화 하기위해서 LFH는 LFH내에서 자주 사용되는 내부 관리용 구조체를 **CPU 코어수 * 2**개의 슬롯(Slots)으로 확장한다.  
그리고 스레드가 LFH에 접근할 때, 이 슬롯 중 1개를 할당하여 접근하도록 한다.  
이렇게 스레드에게 슬롯을 할당해주는 작업은 LFH의 affinity manager 컴포넌트가 수행한다.  
affinity manager가 특정 슬롯에 경합이 발생할 것으로 예상되면 스레드에게 다른 슬롯을 할당하여 경합이 없도록 한다.  

## 큰 크기의 메모리 할당
Heap에서 대략 16KB ~ 512KB 크기의 메모리를 할당받으려 할 경우, Heap 내부의 Free List에서 적당한 크기의 블록을 찾아 할당해준다.  
대략 512KB 이상 크기의 메모리를 할당받으려 한다면 VirtualAlloc 함수로 가상메모리를 직접 할당하여 전달해준다.
